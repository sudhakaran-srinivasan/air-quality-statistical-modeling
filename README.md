# Air Quality Statistical Modeling

Statistical modeling and air quality prediction project using the Beijing Multi-Site Air Quality dataset.

## Team Members

- Nina Zhao
- Pravin Suranthiran
- Sudhakaran Srinivasan

## Objective

This project analyzes relationships between meteorological variables and air pollutants such as PM2.5, PM10, NO2, SO2, CO, and O3 using statistical and probabilistic techniques learned in Applied Probability and Statistics for AI.

## Topics Covered

- Descriptive Statistics
- Probability Distributions
- Sampling Distributions
- Confidence Intervals
- Hypothesis Testing
- Correlation Analysis
- Regression Modeling

## Tools & Technologies

- Python
- Pandas
- NumPy
- SciPy
- Matplotlib
- Jupyter Notebook

## Dataset

Chen, S. (2017). *Beijing Multi-Site Air Quality* [Dataset]. UCI Machine Learning Repository. [https://doi.org/10.24432/C5RK5G](https://doi.org/10.24432/C5RK5G)
